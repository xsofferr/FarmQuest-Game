# Техническое задание: FarmQuest

## 1. Общее описание
FarmQuest — это веб-игра «ферма», в которой игрок управляет участком земли, сажает и собирает урожай, ухаживает за животными и выполняет простые мини-игры (например, рыбалка или «поймай морковку»), чтобы получать внутриигровую валюту и развивать ферму.

## 2. Тип проекта
Веб-приложение с клиентской частью в браузере и серверной частью на Node.js.

## 3. Инструменты и стек технологий
- Frontend: HTML, CSS, JavaScript (без фреймворков на этапе MVP)
- Backend: Node.js, Express
- База данных: SQLite
- Система контроля версий: Git, GitHub

## 4. Основной функционал MVP
1. Регистрация и вход пользователя.
2. Экран фермы с сеткой полей (посадка и сбор урожая).
3. Мини-игры (минимум одна: рыбалка или «поймай морковку»).
4. Внутренняя экономика: монеты, покупка семян/улучшений.
5. Сохранение прогресса игрока в базе данных.

## 5. Нефункциональные требования
- Работа в актуальных версиях браузеров.
- Простая адаптация под мобильные устройства.
- Код проекта хранится в публичном репозитории GitHub и разрабатывается через Issues, ветки и Pull Request.
