# Техническое задание: FarmQuest

## 1. Общее описание
Веб-игра "ферма" где игрок:
- Сажает культуры, ухаживает за животными
- Играет в мини-игры
- Прокачивает ферму, зарабатывает монеты

## 2. Тип проекта
**Веб-приложение** (SPA с сервером)

## 3. Стек технологий
Frontend: HTML/CSS/JavaScript (Vanilla)
Backend: Node.js + Express
База: SQLite
Хранилище: JSON + БД игроков
Деплой: Vercel/Netlify + Railway


## 4. Основной функционал MVP
1. **Регистрация/авторизация** (локальная БД)
2. **Главный экран фермы** - сетка полей (посадить/собрать)
3. **Мини-игры** - 2 простые (рыбалка, торг)
4. **Экономика** - монеты, магазин семян
5. **Прогресс** - уровни фермы

## 5. Архитектура
public/ - статика (HTML/JS/CSS)
src/
routes/ - API endpoints
db/ - SQLite схемы
game/ - игровая логика

## 6. Нефункциональные требования
- Адаптивный дизайн (mobile-first)
- Локализация RU/EN
- Производительность: 60 FPS мини-игры
