# FarmQuest - Ферма с мини-играми

## Описание проекта
Веб-приложение - фермерская игра с мини-играми: посадка/сбор урожая, рыбалка, торговля. Игрок развивает ферму, зарабатывает монеты, открывает новые культуры и животных.

**Тип проекта**: Веб-приложение  
**Цель**: Демонстрация командной разработки с Git workflow
